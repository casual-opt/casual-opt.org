# Casual Optimization

## About us

Casual Optimization は、特定のベンダーに依存しない数理最適化のコミュニティです。ビギナーにも専門家にも有用な情報の集約や提供を通じた数理最適化の普及を目指しています。数理最適化に興味のある方全員に向けた、オープンなコミュニティです。専門家の方もビギナーの方も、ただ興味のあるだけの方も、是非お気軽にご参加ください！みんなで盛り上げていきましょう！
